#  Smart Garbage Reporting System


##  Overview

The **Smart Garbage Reporting System** is a web-based application designed to tackle the issue of waste mismanagement in urban areas. It empowers citizens to report overflowing bins and uncleared garbage spots in real-time using geo-tagging. The platform provides a centralized dashboard for municipal authorities to track, assign, and resolve these issues efficiently, ensuring a cleaner and healthier environment.

##  Features

###  For Citizens (User Module)
* **Easy Reporting:** Capture and upload images of garbage with automatic GPS location tagging.
* **Real-time Tracking:** Monitor the status of reported issues (Submitted -> In Progress -> Resolved).
* **History:** View a log of all past reports and their outcomes.

###  For Authorities (Admin Dashboard)
* **Live Map Visualization:** View all reported issues on an interactive map to identify high-waste zones.
* **Task Assignment:** Assign specific cleanup tasks to sanitation workers or drivers.
* **Verification:** Validate cleanup via "After" images uploaded by workers before closing a ticket.
* **Analytics:** Generate reports on response times and waste volume.

###  For Sanitation Workers
* **Job List:** View assigned tasks with location details.
* **Navigation:** Integrated map directions to the waste spot.
* **Proof of Work:** Upload images of the cleared area to complete the job.
